ComputerAvenue is a modern e-commerce web application designed for tech enthusiasts. It provides a seamless shopping experience for purchasing the latest gadgets, laptops, accessories, and more. With an intuitive user interface, secure payment options, and a well-organized product catalog, ComputerAvenue ensures a hassle-free online shopping experience.

## Features

- **Customer's Page**: Browse top-selling devices and explore a wide range of computers, laptops, cameras, phones, and peripherals.
- **Order Management**: Easily add products to the cart, cancel orders, and view order details for a seamless shopping experience.
- **Admin Dashboard**: Gain insights through data analysis, tracking sales, customer activity, and inventory.
- **Product Management**: dmins can add, delete, view, and edit products to keep the catalog up to date.
- **Order Control**: Manage customer orders efficiently, including order status updates and cancellations.
- **Customer Management**: View and manage customer details, ensuring smooth interactions and support.

## Tech Stack

- **Frontend**: Built with **HTML**, **CSS**, and **JavaScript** for a responsive and interactive user experience.
- **Styling**: Designed using **Bootstrap** for a responsive layout and **SCSS** for modular and maintainable styling.
- **Templating Engine**: Leveraged **EJS (Embedded JavaScript)** for dynamic HTML rendering on the server-side.
- **Backend**: Powered by **Node.js** for handling server-side logic and API requests.
